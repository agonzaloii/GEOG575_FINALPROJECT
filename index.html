OSM USA Changeset Viewer

This visualization depicts the changes done to OpenStreetMap in the United States of America.Each change is noted as a changeset and can be anything from an addition,deletion, and/or modification of a feature within the map. Data was taken using overpass API and openstreetmap. Click drop-down menu above to begin exploring.You can view select features on the dropdown on the left and the timeframe on the right.
